
# Telegram Bot Starter

  ### Based on the [vk-bot-starter](https://github.com/xzeldon/vk-bot-starter) template by [@xzeldon](https://github.com/xzeldon) and [Puregram](https://https://github.com/nitreojs/puregram) library

## Setup


```

npm install

```

  

  

## Development


```

npm run dev

```

  
## Production

  
```

npm run start

```
